<form> Attributes

Attribute Name Description Reference
accept-charset="" Accept Character-Set specifies the character encodings that are to be used for the form submission. W3Schools
action="" Action Specifies where to send the form-data when a form is submitted W3Schools
autocomplete="" Auto Complete Set to "on"/"off". The browser "may"/"may not" auto complete the entries. W3Schools
method="" Method Assigns a HTTP verb to the HTTP request. W3Schools
name="" Name Specifies the name of the form. W3Schools
novalidate="" No Validate It specifies that the form-data (input) should not be validated when submitted if present. W3Schools
rel="" Relation Specifies the relationship between the current document and the linked document. W3Schools
target="" Target Specifies where to receive the response that is received after submiting the form. W3Schools
